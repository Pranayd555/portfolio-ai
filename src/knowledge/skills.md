# Technical Skills Matrix

## Frontend Development
* **Core Frameworks:** Angular (v4 to v21), React 19, React.js
* **State Management:** NgRx (Store, Effects, DevTools), Redux, RxJS.
* **Languages:** TypeScript, JavaScript (ES6+) HTML5.
* **Styling & Layout:** Tailwind CSS, Angular Material, Bootstrap, CSS/SCSS.
* **Interactive Web Graphics:** Three.js, HTML5 Canvas, Custom Shaders
* **Design Systems:** Figma MCP, UI Design Systems.

## Backend & API Engineering
* **Runtime Environments:** Node.js (v22, pinned to `22.17.0` for environment locking).
* **Frameworks:** Express.js.
* **API Architectures:** REST APIs, WebSockets (Socket.io, native WS).
* **Templating Engines:** Handlebars.js (compiled template caching optimizations).
* **Session & Auth Architecture:** OAuth2 (Google, LinkedIn), Role-Based Access Control (RBAC), Secure Impersonation Flows, JWT Sliding Expiration Sessions.

## Database & Cloud Infrastructure
* **NoSQL Databases:** MongoDB Atlas, Mongoose ODM (Atomic transactions, multi-tab session handling).
* **Object Cloud Storage:** Amazon AWS S3, Cloudflare R2 (Presigned secure URL mechanics, programmatic SDK bindings).
* **Media & CDN Optimization:** Cloudinary CDN.
* **Web Servers & Gateways:** Nginx, InterServer VPS.
* **Deployment & Platform PaaS:** Docker, Render, Vercel, Cloudflare Pages.
* **CI/CD Build Pipelines:** GitHub Actions, Custom Node.js stream packaging architectures (`archiver`).

## Software Engineering Practices
* **Testing Suites:** Jasmine, Karma.
* **Build Systems:** Webpack 5, UMD custom builds, npm scripts.
* **Engineering Standardizations:** Accessibility (A11y WCAG compliance), Internationalization & Localization (i18n & i10n), Code Refactoring, Production Tracing (Winston logging).
